# Kito Mart - Company Profile Website

A professional company profile website for Kito Mart with multiple pages and responsive design.

## Project Structure

```
.
├── index.html              # Home page
├── about.html              # About us page
├── services.html           # Services page
├── contact.html            # Contact page
├── css/
│   ├── style.css          # Main stylesheet
│   ├── about.css          # About page styles
│   ├── services.css       # Services page styles
│   └── contact.css        # Contact page styles
├── img/
│   ├── market/            # Market related images
│   └── profile/           # Profile images
└── js/
    ├── main.js            # Main JavaScript functionality
    ├── filter.js          # Filtering functionality
    └── form-validation.js # Form validation scripts
```

## Features

- Multiple pages for company information (Home, About, Services, Contact)
- Responsive design with custom CSS styling
- Form validation for contact forms
- Filtering functionality
- Product/market images

## Getting Started

1. Open `index.html` in your web browser to view the website
2. Navigate through the site using the menu links
3. Fill out forms with proper validation

## Files Overview

### HTML Pages
- **index.html** - Landing page and main entry point
- **about.html** - Company information and background
- **services.html** - Products and services offered
- **contact.html** - Contact form and information

### JavaScript
- **main.js** - Core functionality and DOM manipulation
- **filter.js** - Product/service filtering logic
- **form-validation.js** - Client-side form validation

### Stylesheets
- **style.css** - Global styles and layout
- **about.css** - About page specific styles
- **services.css** - Services page specific styles
- **contact.css** - Contact page specific styles

## Requirements

- Modern web browser (Chrome, Firefox, Safari, Edge)
- No dependencies or build tools required - pure HTML/CSS/JavaScript
